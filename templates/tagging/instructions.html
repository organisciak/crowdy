Label each of the images below with at least two tags. A tag is a short word or phrase describing what is in the image.

This task is timed: tag as many items (minimum 2) as you can in {{time.limit}} seconds. We'll give you bonuses for each extra image after the second.

The purpose of this task is to help people find these images. Thus, one way to think of good tags is to consider, if somebody was searching for this word, would this be an image that they'd want to find?

Good and interesting tags describe actions and objects in the image. Tags can be multiple words, and should stand alone.

Bad tags don't describe what is in the image, describe things too generally, or describe things that are not the focus of the image. For this task, personal tags are not helpful.
